# ✈️ Air Plane Fight - 飞机大战小游戏

> 大一C语言项目 | 基于EasyX图形库

---

## 🎮 游戏简介

这是一个用C语言和EasyX图形库编写的飞机大战小游戏。虽然还是个半成品，但已经实现了基本的游戏功能和界面控制。

---

## 🕹️ 操作说明

| 操作 | 按键 |
|------|------|
| 移动 | **WASD** |
| 发射子弹 | **鼠标左键** |
| 暂停/继续 | **ESC** |

> 💡 **提示**：开始界面按任意键进入游戏

---

## 🛠️ 编译与运行

### 环境要求
- Windows系统
- Visual Studio 2017 或更高版本
- EasyX图形库

### 安装EasyX
1. 访问 EasyX官网：https://easyx.cn/
2. 下载对应Visual Studio版本的安装包
3. 运行安装程序，自动配置到VS中
